位运算

用途

（1）n的二进制表示中第k位是几（从个位开始算）
（2）返回x的最后一位1 (lowbit)

思路

（1）n >> k & 1

  • 先把第k位移到最后一位 n>>k
  • 看个位是几 x&1

（2） lowbit(x)=x&-x

原理：
x&-x = x&(~x+1)
eg:
x = 1010100
~x = 0101011
~x+1 = 0101100
x&(~x+1) = 0000100

    区间合并

    用途

    快速合并两个有交集的区间

思路

（1）按区间左端点排序
（2）设当前区间左端点为st1，右端点为ed1

  • st2 >= st1&&ed2 <= ed1 → 不变
  • st2 >= st1&&ed2 > ed1→ ed1 = ed2
  • st2 > ed1 → 当前区间放入结果中，新的区间为(st2,ed2)
