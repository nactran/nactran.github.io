热国固穷

1、新加坡、马来西亚的增长
2、地理大发现前低纬度的印加、阿兹特克比温带的北美洲印第安人发达

财富的逆转与地理无关，与被殖民的方式有些相关 => 殖民者施行的制度造成了“财富逆转”

对了解世界不平等既有用也没用。有用：文化->制度差异 时

奴隶贸易 -> 破坏产权和人权的制度 ->非洲人彼此互信度低 （有论文支撑：Nathan Nann and Leonard Wantchekon (2011). The Slave Trade and the Origins of Mistrust in Africa）

第三章

朝韩差距：不能用文化、地理来解释，只能用制度来回答
汲取性经济制度 VS 包容性经济制度
包容性经济制度：安全的私人产权是核心 被大多数人拥有
反例——巴巴多斯：有安全的私人产权，但只有少数人拥有
汲取性经济制度：从社会一部分人那里攫取财富，让另一部分人受益

包容性经济制度

  • 创造包容市场，提供平等舞台 => 提高效率
  • 鼓励技术 （保护产权，创造公平环境允许新企业进入）
  • 鼓励教育（教育回报）

多元主义和包容性经济之制度间存在联系，但还需要国家集权：
缺乏国家集权的多元主义：索马里

包容性政治制度：集权+多元化

汲取性政治制度：上述条件不满足

汲取性经济制度与汲取性政治制度相伴
汲取性政治制度：权力掌握在少数精英手中
汲取性政治制度下，精英构建汲取性经济制度致富，他们的经济财富和权力又帮助他们巩固政治优势

包容性经济制度是在包容性政治制度的基础上形成的，包容性政治制度把权力广泛分散于社会，并限制滥用。
包容性政治制度下汲取性经济制度是不可能长期存在的
包容性经济制度既不支持汲取性政治制度，也不被汲取性政治制度支持

为什么不选择繁荣：
经济增长既有赢家也有输家：创造性破坏
对创造性破坏（成为经济增长输家）的担心成为反对包容性经济制度和政治制度的根源

政治制度没有理由自动多元化
任何社会当中存在建立更集权的国家制度的激励 =>
为什么索马里还是没有集权制度？
试图集权会引起其他宗派、集团或个人的愤怒，因为他们会成为政治输家 => 阻碍、扰乱

刚果：
没有改革经济制度的激励：改革会使精英经济受损，经济和政治上失势
没有运用权力建立政权集权，因为这样做会和刺激经济增长一样带来同样的抵制和政治挑战

汲取性政治制度下的增长：

  • 精英能够直接将资源配置到自己控制的高生产率活动中时：
    • 加勒比群岛种植园制度
    • 苏联五年计划：资源从农业领域强制转到工业领域
  • 当制度允许某种程度的包容性经济发展
    • 精英阶层的地位足够安全，确信转向包容性经济制度不会威胁到他们的政治权力时，可能允许进行某些变化
    • 由于历史情形造成汲取性政治制度拥有了较为包容的经济制度，而且统治阶层决定不去阻碍它：
      韩国——受美国支持、有较包容的经济制度 => 朴正熙时代经济增长

没有政治集权的国家：实现有限的增长都面临困难

第四章

黑死病在西欧和东欧带来的不同影响：
西欧（英国）：
工资价格上升 => 贵族试图抑制价格和劳动力流动，失败 => 包容性劳动力市场的出现
东欧：
工资价格出现上升趋势 => 贵族强化对农民的控制 => 第二次农奴制
微小差别：东欧领主组织得更强一些，农民组织得更差一些……

英国和西班牙的不同道路：
1688年：英国政治制度比西班牙更多元、包容
1588年：没有太大差别 专制君主与有征税权的议会间斗争
微小的差别：
西班牙：
国王拥有美洲帝国，不再依赖议会
国王垄断美洲贸易
英国：
国王财政独立性差，依赖议会 => 议会最终胜利
国王无法垄断美洲贸易 => 大西洋贸易和殖民创造出一群与皇室联系不大的富裕商人 => 专制主义反对者力量更强大

关键节点上细小差别引发不同反应，差异的累积引发制度漂移
黑死病 => 西欧封建制度解体；东欧第二次农奴制
大西洋贸易 => 英国政治制度多元化；西、法专制加强

历史的偶然道路：
1588英国战胜西班牙无敌舰队 => 英国大西洋贸易进一步发展 => 1688政治变革
假如英国战败？（会有假如吗，凭什么？）

世界各地的情况：
刚果vs博茨瓦纳
清国vs日本
奥斯曼帝国

快速经济增长的结束：
1、 偏离了包容性制度
2、 汲取性制度增长的结束（苏联、阿根廷）

立论：

抓住关键节点
打破现有模式
改革现行制度
走向繁荣道路

第五章

苏联的增长：
通过重新配置劳动力和资本积累实现（把资源配置到高效率的工业部门）
苏联的激励：
分红制度：难收成效
创新面临生产目标达不到、红利得不到支付的风险；产量目标通常建立在之前产量水平的基础上 => 工人不愿扩大产量
只要政治权威和权力还留在苏共手中，就不可能从根本上改变人们面对的基本激励。

制度创新可以巩固汲取性制度，使增长成为可能。
汲取性制度下增长受限制：
1、缺乏技术进步的激励
2、利益的争夺带来不稳

卡塞河两岸：
乐乐人vs布松人 文化相似，乐乐人穷，布松人富
政治革命 -> 集权国家 -> 组织生产 -> 汲取性制度下的增长

长夏（新石器革命的起源）：
定居生活：更容易起冲突，需要规则维护 ＜＝＞ 将权力集中
（谁先谁后？）
那图夫人：先形成等级社会，再进入农业社会
结论：政治人物集中权力 -> 推动定居 -> 进入农业社会
贾雷德・戴蒙德：物种驯化 -> 农业革命、定居生活 -> 等级社会
阿西莫格鲁：制度变迁 -> 定居 -> 强化制度变迁 -> 新时期革命

不稳定的汲取（玛雅城邦）：
天赐君主(divine lord) -> 大型工事、社会分工 -> 经济增长
|-> 战争 动荡 -> 覆灭

第六章

威尼斯：由包容转向汲取
包容的契约创新：康曼达契约(commenda)
开放的政治制度：
总督选举、受公爵议会监督
大议会的形成、公爵议会的权限

包容性的制度 -> 创造性破坏：
（政治上）通过康曼达契约致富的商人会挑战现有政治人士的政治经济利益

既得利益者试图排斥新人：逐渐加强大议会成员资格的限制（13世紀末） -> 形成世袭贵族 ->政治上转向汲取
废除康曼达契约的使用、贸易国有化（14世纪初） -> 经济上转向汲取

结局：威尼斯衰落

罗马的兴衰
罗马共和国：
有多元主义成分的政治制度
共和国的经济增长及证据
不可持续的增长：元老阶级VS平民 土地兼并加剧、贫富差距扩大
平民与贵族矛盾扩大：
提比略（Tiberius Gracchus）、盖约（Gaius Gracchus）改革 -> 被元老阶级杀害
苏拉（代表元老）VS凯撒（代表平民）-> 凯撒之死->
安东尼VS屋大维 -> 向罗马帝国转型->
包容性制度 => 汲取性制度
罗马帝国的衰亡：
汲取性制度下利益集团内部为了争权夺利讨好外族指挥官，引狼入室
罗马衰亡的原因：
汲取性制度下不允许创造性破坏的出现；奴隶制的盛行使多数人缺乏创新的动力
（弦外之音：只有允许源源不断的创造性破坏，才能实现经济的持续增长。这只能在包容性制度下实现）

第七章

转折点——英国的道路
汲取性制度：不允许创造性破坏 -> 新时期革命和工业革命之间生活水平没有持续提高的主要原因

英国的政治冲突：

  • 1215 <大宪章>
  • 1265 第一次选举议会：
    迈向多元化的第一步：英国的议会：除了代表封建贵族的利益，还代表不同行业小贵族的利益（包括商人、富农）
  • 1381 农民起义：政治权利不再简单地从国王再分配给地主，而从精英还要再分配给群众
  • （15世纪末）玫瑰战争=>都铎王朝：
    • 政治集权的增强：削弱贵族权力、政府从国王私人机构变为制度性机构（官僚国家）、剥夺教会权力
    • 政治上的集权推动贵族与国王争夺对集中的国家权力的控制
  • 1603 詹姆士一世 斯图亚特王朝： 专制、集权、垄断
  • 1623 <垄断法>禁止詹姆士一世制造新的国内垄断（仍可以授权国际贸易垄断）
  • 查理一世时期：
  • 1642 内战爆发 获得垄断权的大商人（保皇派） VS 反对垄断的手工业者（议会派)
  • 1688 光荣革命 权威和决策权转到议会
  • 1689 权利法案 确立宪政原则

制度变化：

  • 产权得到保护
  • 政治制度出现多元主义性质： 民意通过请愿传达给议会
  • 逐渐废除垄断权
  • 再分配税收负担 重组产权
  • 金融革命：保障普通商人得到贷款
  • 议会继续政治集权的过程、官僚机制

工业革命

  • 国内垄断的废除
  • 更安全、更有效率的产权制度支持创新者和企业家
  • 议会制度和请愿机制：不同利益集团通过请愿/投票参与决策

1736 曼切斯特法案 保障棉纺织业发展
工业发展带来创造性破坏
棉纺业摧毁羊毛纺织业
工业资产阶级、中产阶级、工人要求更多权利：
“废除谷物法” “普选权” “（不记名）投票选举”
1832 议会改革
1846 谷物法废除

为什么是在英格兰：
包容性制度：鼓励创新者进入，阻止排挤和垄断
宪政

第八、九章

为什么落后？直接原因：没能抓住工业革命带来的技术变革
发展的障碍
不保护产权、王室垄断阻碍工商业发展：
西班牙
统治者担心创造性破坏，阻碍商业发展和工业技术引入：
奥匈帝国
俄罗斯
中国
埃塞俄比亚
没有形成集权国家：
索马里

并不是不愿意发展，而是在巩固政权和发展之间选择了前者

逆转发展
荷兰在东南亚殖民扩张：通过强制实施或进一步强化现存汲取制度，导致东南亚国家落后
奴隶贸易：推动英国走向包容；却强化了非洲国家的专制制度。战争和奴隶贸易最终破坏了撒哈拉以南非洲存在的所有秩序和权威。
1807 废除奴隶制 => 黑奴无法再卖给欧洲人，在非洲当地从事强制性的利润丰厚工作（出口转内销）
二元经济：
早期英国南非殖民：通过削弱首领的权力削弱了当地汲取性制度和专制主义制度，当地经济发展
后期（20世纪）：只给黑人分少量土地，使其贫困，迫使黑人劳动力转移向工矿业、增强首领权力 => 汲取型制度强化

第十、十一章

繁荣的扩散
澳大利亚、美国：
当地资源和原住民人数不允许殖民者实行像西班牙在拉美实行的汲取性制度 => 殖民者没办法压榨原住民，只能激励移民 => 逐步建立包容性政治经济制度
另一个优势：和英国相比，没有封建主义传统

法国大革命 拿破仑法典：法国输出革命，打破中西欧旧有封建制度
日本明治维新：主动向西方学习
