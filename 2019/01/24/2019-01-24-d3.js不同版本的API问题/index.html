2019-01-24-d3.js不同版本的API问题

一、undefined is not an object (evaluating ‘d3.scale.linear’)

原因：在d3js v4版本以后中的线性比例尺不再使用d3.scale.linear()。而是使用d3.scaleLinear()来代替它。 同理，d3.scale.ordinal()要改为d3.scaleOrdinal()
